# MCM Reference

Quick reference to every page in the SexLab Survival MCM, in the order they appear, with what each controls and a link to the detailed page where one exists.

!!! tip "Configure before playing"
    Set your options on a fresh save before diving in — some systems assume you've reviewed their settings. See [Getting Started](getting-started.md) for first-launch notes.

    - **Settings** holds global keys, import/export, and general options.
    - **Interfaces** is where mod integrations are toggled — see [Mod Integrations](integrations.md).
    - **Stats & Info** and **Stats & Info 2** are read-only status pages; they display current state and don't change settings.
